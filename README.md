# StockWatch

### A demo is worth a 1000 words

Tech Stack : TypeScript , NextJS, NodeJS , Express , TailwindCSS, Sequelize, SQLite

## Setup

```
git clone
cd server
```

add env for server

```
PORT=8000
JWT_SECRET=<"secret token for jwt">
```

then

```
yarn && yarn start
```

```
cd client
```

add env for client

```
NEXT_PUBLIC_BACKEND='<backend_url>'
```

then

```
yarn && yarn start

```
